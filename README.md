# Medusa-the-Origin
原灵深海遗梦的移植项目源码

## 分支说明
如你所见，主分支干净的像阿草的银屯，文件是按分支发布的  
`site`分支是本项目网页的最终生成文件，可直接部署在服务器上  
`android`分支是本项目安卓移植所需要的代码，由于安卓移植并没有对资源文件进行大修改（就是把视频转码为webm了，并压缩了图片），所以不提供资源文件，节约仓库空间  
`web`分支是本项目网页移植所需要的代码，一样没有提供没有变化的资源文件，但新增了两张图片资源，已附带
